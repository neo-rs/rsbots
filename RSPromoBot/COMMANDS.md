# RSPromoBot Commands

## Slash Commands

### `/promo_dm`
Open the Promo DM Builder.

Flow:
- Select Role
- Edit Message
- Edit Settings
- Preview
- Test Send
- Start Campaign

### `/promo_status`
Show the current or latest campaign status.

### `/promo_pause`
Pause the active promo campaign. (Staff only.)

### `/promo_resume`
Resume a paused promo campaign. (Staff only.)

### `/promo_cancel`
Cancel the active promo campaign. (Staff only.)

### `/promo_history`
Show recent campaign history.
